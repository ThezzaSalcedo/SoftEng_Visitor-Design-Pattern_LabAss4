# SoftEng_Visitor-Design-Pattern_LabAss4
## Furniture Shipping Cost Calculator

### Overview

This project demonstrates the implementation of the Visitor Design Pattern in Java to calculate shipping costs for different types of furniture in an e-commerce platform.

The platform sells various furniture items such as chairs, tables, and sofas, each with its own shipping cost calculation logic. 

## Problem Statement

In an e-commerce furniture platform, different furniture items may require different shipping cost calculations:

* Chairs are lightweight and use a flat shipping rate depending on size.

* Tables calculate shipping based on weight.

* Sofas calculate shipping based on delivery distance.
